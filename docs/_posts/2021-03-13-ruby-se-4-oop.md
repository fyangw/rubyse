## Ruby SE オブジェクト指向

主に比較となる言語はCOBOL、VB(.net)、Java 8、JavaScript、Python 3。

機能         |Ruby          | COBOL     | VB(.net)          | Java 8              | JavaScript             | Python 3
-------------|-------------|------------|------------------|---------------------|-------------------------|------------------------------------
クラス       | `class...`   |  -        | `class...`       | `class...extends...{...}` | `function...{...}`| `class ... : ...`
メソッド     | `def ...`    |  ''       | `Function ...`   | `... ...(...){...}`       | `function...(...){...}`| `def ...: ...`
